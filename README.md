# 🚀 Employee Onboarding & Integration Platform

**An Interactive Platform for New Employee Adaptation and Integration**
This is a practical university project aimed at simplifying, automating, and adding interactivity to the onboarding process for new hires.

---

## 🎯 Problem Statement
The system addresses common challenges faced during employee adaptation, such as:
* Scattered administrative tasks.
* Information loss and missed deadlines.
* Difficulties in integrating new team members.

## 👥 User Roles (Spring Security)
1. **Admin (IT Administrator):** Manages technical provisions, accounts, and system access.
2. **HR Manager:** Manages the employee database and monitors the overall integration process.
3. **User (New Employee):** Tracks personal onboarding progress and completes assigned tasks.

## 🌟 Key Features
* **CRUD Operations:** Full management of employees, departments, and onboarding tasks.
* **Status Tracking:** Task execution monitoring (Pending, In Progress, Done).
* **Gamified Progress Board:** Visual progress bar for new employees to track their journey.
* **Mood Monitoring (Feedback):** Daily check-ins to collect quick feedback from new hires.
* **Mentor Pairing Module:** Automatic assignment of experienced colleagues to assist new employees.

## 💻 Tech Stack
* **Backend:** Java, Spring Boot, Spring Security
* **Database:** PostgreSQL, Spring Data JPA (Hibernate)
* **Frontend:** Thymeleaf, HTML/CSS
* **Architecture:** Monolithic (Standard libraries, no external APIs)
